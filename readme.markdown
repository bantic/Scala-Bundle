## Tristan's Scala Bundle ##

I fixed some multiline string handling where the syntax highlighting got
messed up with `"""` quotes. It was probably the bundle by [Michael
Nischt](http://gestalt.monoid.net/blog/2008/02/new-scala-bundle-for-textmate.html),
which I was generally pretty happy with.

### Use Me

Double click the `Scala.tmbundle` directory in finder.
